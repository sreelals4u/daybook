📖 HOW TO USE — DAYBOOK PRO
Daily workflow guidance
🌅 Morning Routine
1
Go to 💰 SUMMARY, tap Today button, enter yesterday's closing balance in the yellow Opening Balance cells.
2
Go to 🏦 DENO, count your vault physically, update the COUNT column with actual notes in hand.
3
Tap 🔄 Reset Movement to zero all IN/OUT for the new day.
📝 Adding Entries — New Agent Without Code
1
Tap ➕ Add Entry. Select a Scheme first — agent field appears or hides based on scheme type.
2
New agent without code? Type the agent name in the Agent Name field. In the Agent Code field type NEW LIA or PENDING. Entry saves normally. Edit later when code is assigned.
3
For existing agents: type 2-3 letters → autocomplete appears → tap to select → code fills automatically.
💰 Daily Closing Balance
1
Go to 💰 SUMMARY tab. Tap Today to select today's date.
2
Enter today's Opening Balance in yellow cells (Cash, Bank, Locker).
3
The Closing Balance row shows the correct end-of-day balance based only on today's transactions.
4
Next morning, this closing becomes the next day's opening balance.
🏦 DENO Daily Workflow
1
Vault Stock COUNT = actual notes physically in hand. Enter by counting, not calculating.
2
Today's Movement IN = notes received from agents. OUT = notes paid out.
3
Running Total = vault after today's movement. This is your closing vault count. Next morning copy these numbers to Vault Stock COUNT.
💾 Data Safety
1
Every evening tap 💾 Backup. Save to Google Drive or WhatsApp to yourself. Data is safe even if phone is reset.
2
If data is lost, tap 📂 Restore and pick your backup file. Everything comes back in one tap.
3
At month end tap 🗓 New Month — auto-saves backup, clears entries, rolls closing → opening balance.
🏠 Install as App on Samsung
1
Upload index.html to GitHub Pages at sreelals4u.github.io/daybook (already done).
2
Open that link in Chrome on your Samsung. Tap ⋮ three dots → Add to Home screen → Add.
3
Data is stored in IndexedDB on your phone. It never leaves your device. GitHub only hosts the app code.
