📖 HOW TO USE — DAYBOOK PRO
Step-by-step daily workflow guidance
🌅 Morning Routine
1
Go to 💰 SUMMARY tab. Enter yesterday's closing balance in the yellow Opening Balance cells — Cash, Bank, Locker separately.
2
Go to 🏦 DENO tab. Physically count your cash vault. Update the COUNT column with the actual notes/coins in hand. Tap away to save.
3
Tap 🔄 Reset Movement to zero all IN/OUT counts for the new day. Vault stock stays unchanged.
📝 Adding Entries
1
Tap ➕ Add Entry on Day Entry tab. Enter the date and Sheet Number from your manual ledger.
2
Select a Scheme/Category first. If the scheme requires an agent (like RENEWAL RD), the agent field appears. If it's a non-agent scheme (like TELEPHONE BILL), the agent field is hidden.
3
For agent schemes: type 2-3 letters of the agent name → autocomplete appears → tap to select → code fills automatically.
4
Enter amounts: Cash IN for cash collected, Bank IN for UPI/IMPS/NEFT, Locker IN for ≥₹1,00,000 cash, Cash PAY for payments out. Leave others blank.
🏦 DENO Daily Workflow
1
Vault Stock COUNT = what you physically have. You enter this by counting notes in hand. Do NOT manually add — just enter the actual count you see.
2
Today's Movement = IN when notes come in from agents, OUT when you pay out. Enter the number of notes, not amounts.
3
Running Total at the bottom = your vault after adding IN and removing OUT. This is your end-of-day vault count. Next morning, this becomes your new Vault Stock COUNT.
4
Use Enter key to jump between IN → OUT → next denomination for fast entry.
💾 Data Safety & Backup
1
Every evening, tap 💾 Backup. Save the downloaded file to Google Drive or WhatsApp to yourself. This protects against browser cache clearing.
2
If you ever lose data, tap 📂 Restore and pick your backup file. Everything comes back exactly as it was — entries, balances, agents, deno counts.
3
Use 📥 Export to get a clean Excel file for sharing with management or archiving monthly records.
4
At month end, tap 🗓 New Month. Enter the new month name (e.g. MAY 2026). The app auto-saves a backup, clears entries, and rolls closing → opening balance.
🏠 Save to Home Screen (Samsung)
1
The app must be served over https:// to be saved to home screen. Open your browser and go to https://telegra.ph, create a page and paste the HTML there — or host on Netlify Drop (drag and drop the file at app.netlify.com/drop).
2
Once open on a real https:// URL: Samsung Internet → tap ☰ menu → Add page to → Home screen. Chrome → tap ⋮ → Add to Home screen.
3
Your data is stored locally on the device via localStorage — it does not go to any server. The hosting is only for the app shell.
